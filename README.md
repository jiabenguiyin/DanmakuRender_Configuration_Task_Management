# DanmakuRender Configuration Task Management—— 一个批量管理DanmakuRender5配置文件的小工具
结合网络上的代码写的一个能录制带弹幕直播流的小工具，主要用来录制包含弹幕的视频流。     
- 可以一键配置启用禁用录制主播
- 设置录制天数
- 可以设置批量主播
    
初衷是DanmakuRender5管理配置文件太麻烦，所以我为了方便自己使用，写了这个小工具，配合DanmakuRender5使用。效果很棒！
